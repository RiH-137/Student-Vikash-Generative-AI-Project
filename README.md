## 🛠️ **Tech Stack**

- **Programming Language:** Python  
- **Libraries:**  
  - Machine Learning Algorithms  
  - Natural Language Processing (NLP)  
  - Generative AI  
- **Framework:** Streamlit  

---

## 🚀 **Features**

- 🖼️ **Image-to-Text Conversion** with Multilingual Support.  
- 🎥 **Note-Taking** from YouTube Transcripts.  
- 📄 **PDF Question Answering**.  
- 💬 **Multilingual Chatbot** Powered by Generative AI.  
- 📊 **Resume ATS Score Analysis** with Recommendations.  
- 🛠️ **Many More Versatile Tools for Students!**  

---

## 📦 **Installation**

To run this project locally, install the required dependencies:

```bash
pip install langchain_text_splitters streamlit Pillow PyPDF2 youtube_transcript_api google.generativeai langchain_google_genai langchain.chains.question_answering langchain.prompts

```
